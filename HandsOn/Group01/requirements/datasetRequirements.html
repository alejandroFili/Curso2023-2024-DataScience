Requirements

R1. Data are in one of the selected smart city domains.

El data set seleccionado está dentro del dominio del Smart traffic , que pertenece al de las ciudades inteligentes(Smart citties). Hemos seleccionado la ciudad de Pittsburgh porque los datos eran compatibles con el planteamiento del proyecto.

R2. Data are available as a CSV file.

Los datos encontrados están disponibles como CSV en la página de Google dataset search .

R3. Data have an open license so they can be published.

El CSV tiene como licencia:  CC0 1.0 Universal Public Domain Dedication .

“You can copy, modify, distribute and perform the work, even for commercial purposes, all without asking permission.”

R4. Data can be easily linked with generic real-world entities (e.g., locations).

Dentro del data set encontramos como entidades la ciudad, las calles y los barrios.

R5. Documentation exists for the data (optional).

Existe una breve descripción en la página web de datos de Pensilvania. 

La página de los datos es: https://data.wprdc.org/dataset/traffic-count-data-city-of-pittsburgh
