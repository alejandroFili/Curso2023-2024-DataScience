Data Source Analysis

Data Overview

London Air Quality Network Camden. Link: "https://www.nationalarchives.gov.uk/doc/open-government-licence/version/3/". BD que contiene datos recopilados por sensores operados por London Air sobre la las características del aire en Camden ( Londres )

Data Schema

El CSV está compuesto por 18 columnas que nos dan datos sobre: Código identificador de sitio, Nombre del sitio, Tipo del Sitio, Fecha de boletín y fecha de boletín formateada, rangos de la calidad del aire en relación con la concentración de dióxido de nitrógeno NO2 y el índice, última captura y última semana, código y nombre del distrito, coordenadas este-oeste y norte-sur, longitud, latitud, grado de precisión en la representación geográfica, última actualización y localización ( coord )

Data Types

En todas nuestras columnas tenemos datos categóricos ( texto ) menos en las coordenadas este-oeste y norte-sur que tenemos reales ( real )

Data Size

Nuestra base de datos está formada por 18 columnas y 70456 filas

Data Range

En las columnas con datos tipo Real, para las coordenadas este-oeste tenemos un valor min de 526630 y max 530527, mientras que en las coordenadas norte-sur el min es 181502 y el max 184392

Data Quality

La mayoría de las columnas nos proporcionan datos limpios ( tidy data ). Sin embargo, aparecen columnas con datos inexistentes o no relevantes como las de última captura y última semana ( todos los valores = 'No' ), y el grado de precisión en la representación geográfica ( todos los valores = 'Unknown' ), o algunos valores de los rangos de la calidad del aire en relación con la concentración de dióxido de nitrógeno NO2 ( 'No Data' )

Data Distribution

Al ser la mayoría de columnas tipo texto, no podemos obtener más estadísticas que las mencionadas anteriormente, mientras que de las dos columnas con valores numéricos podemos añadir que el promedio para las coord este-oeste es de 529117 y para las norte-sur 182646

Data Relationships

Podemos observar claramente como todas las columnas guardan una relación entre sí, cómo es el ejemplo del código y nombre del sitio, o del barrio/distrito, latitud/longitud, coordenadas, los rangos de la calidad del aire en relación con la concentración de dióxido de nitrógeno NO2 y sus respectivos índices, la fecha normal y formateada del boletín y el tipo del sitio junto al nombre y código. Estas relaciones son bastante obvias, pero aún así podemos visualizar una relación entre todas y cada una de las columnas, nada más con su distribución ( cada columna añade información al contexto que teníamos en la anterior, aunque no hablen exactamente de lo mismo ).

Data Context

Estos datos fueron recogidos por sensores situados en varias zonas de varios distritos de Camden ( Londres ) con el fin de recoger las características del aire cómo son la concetración de NO2 y el índice.
